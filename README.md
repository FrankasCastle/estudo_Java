Java – Exercícios do W3Schools

Este repositório contém exercícios de **Java** que fiz acompanhando o tutorial do **W3Schools**.

O objetivo foi aprender e praticar os fundamentos da linguagem Java, passando pelos conceitos básicos até estruturas mais comuns do dia a dia. Cada arquivo representa um exercício ou exemplo prático relacionado a um tópico do tutorial.

Conteúdos praticados

Os exercícios cobrem os seguintes temas:

- Introdução ao Java e estrutura básica de um programa
- Sintaxe
- Impressão de valores no console
- Comentários
- Variáveis
- Tipos de dados
- Type casting
- Operadores
- Strings
- Math
- Booleanos
- Condicionais (if / else)
- Switch
- Laços de repetição (while e for)
- Break e continue
- Arrays

Organização dos arquivos

Os arquivos estão nomeados de forma sequencial (`Exercicio01`, `Exercicio02`, etc.), seguindo a ordem dos tópicos estudados no tutorial do W3Schools.

Cada arquivo `.java` contém um exemplo ou exercício focado em um conceito específico, facilitando a revisão e o entendimento do conteúdo.

Objetivo do repositório

Este repositório serve como:

- Registro do meu aprendizado em Java  
- Prática dos conceitos básicos da linguagem  
- Material de consulta para estudos futuros  

Observação

Este repositório tem fins educacionais e acompanha meu progresso durante o estudo de Java através do W3Schools.
